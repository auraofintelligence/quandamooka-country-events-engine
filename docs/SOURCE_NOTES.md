## Calendar classification

The public calendar is now treated as an event atlas rather than a simple date list.

- `confirmed` means the event had a current public source with a date checked during the 22 May 2026 source pass.
- `past` means the event is kept as public memory and simulation evidence, not as an upcoming listing.
- `tbc` means the event or pattern appears useful but the current date needs a direct organiser/source refresh.
- `recurring` means the record describes a repeated local rhythm that should be confirmed before being promoted.

The Redland City Council What's On calendar is included as a cross-check source, but the listing is embedded through an external calendar widget. Individual event records should still be checked against ferry, organiser, cultural, ticketing, club or venue sources before publication.

## Update rule

Before public launch, refresh:

- dates for all upcoming events
- whether past event records should become public memory pages
- current venue hire details
- council permit links and requirements
- parks and beach access rules
- ferry and bus timetables
- QYAC and cultural authority pathways
- supplier and artist consent
- public notice approval owners
