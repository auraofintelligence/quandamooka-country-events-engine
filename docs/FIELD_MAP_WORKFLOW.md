# Adding field notes from a phone

You do not need to survey the whole island. Add observations whenever you are already near a place.

1. Open the Places map on your phone and tap **Add pin**.
2. Tap **Use my GPS**, or choose **Tap map** if the object is visible away from you.
3. Add a short name and choose what you observed: a new place, a correction, something closed, or an asset check.
4. Add a note and an optional photo. The map shrinks the photo before saving it.
5. Tap **Save observation**. It stays on that phone and is not uploaded anywhere.
6. Tap **Export** when you are ready, then attach the downloaded JSON file when asking Codex to update the permanent map.

The exported file includes the coordinates, reported GPS accuracy, capture time, notes and any attached photos. Export regularly if collecting many photos because phone browsers have limited local storage.

Stay on public land, do not photograph people without permission, and only use the map while safely stopped.
